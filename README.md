# Sales-Performance-Analysis
# Sales Performance Analysis Using Python

## Project Overview

This project analyzes company sales data to discover valuable business insights related to:

- Sales performance
- Profitability
- Customer behavior
- Product performance
- Regional sales trends
- Sales channels
- Discount impact

The project demonstrates a complete data analysis workflow using Python.

---

# Objectives

The main objectives of this project are:

- Analyze total sales and profit
- Identify top-performing regions
- Evaluate sales representative performance
- Analyze product categories
- Discover monthly sales trends
- Study customer purchasing behavior
- Measure discount impact on profitability

---

# Dataset Features

The dataset contains the following columns:

- Product_ID
- Sale_Date
- Sales_Rep
- Region
- Sales_Amount
- Quantity_Sold
- Product_Category
- Unit_Cost
- Unit_Price
- Customer_Type
- Discount
- Payment_Method
- Sales_Channel

# Key Insights

- West region generated the highest sales
- Online sales channel performed best
- Electronics category produced highest profits
- Heavy discounts reduced profit margins
- Returning customers generated more revenue

---

# Business Recommendations

- Increase focus on high-performing regions
- Optimize discount strategy
- Invest more in online sales channels
- Promote high-margin products
- Reward top-performing sales representatives
